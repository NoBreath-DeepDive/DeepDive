## 📚 스터디 방법

1. GitHub 리포지토리를 클론하고 개인 브랜치 생성
2. 각자 정리한 내용을 `.md` 파일로 작성 후 Pull Request
3. 발표자는 해당 주차 발표자료(PPT, 이미지 등) 업로드
4. 리뷰 및 피드백은 PR에 코멘트로 진행
5. 매주 월요일 기준으로 다음 주차 내용 배정
